25.what is GIT vs GITHUB

It was created by Linus Torvalds in 2005 and is open source, meaning its source code is freely available for modification and use. Git enables developers to access their code files from different computers, allowing for collaborative work on projects.
GitHub:- on the other hand, is a platform that hosts code repositories and utilizes Git for version control. It acts as a hub where developers can collaborate on software projects, share files, and manage different versions of their code. GitHub simplifies the process of working with Git by providing a user-friendly interface for hosting and version control.
